## 掲示板アプリ
ユーザー登録・ログイン・投稿・削除ができます。

## 使用言語
- Python
- Flask
- Flask_SQLAlchemy
- SQLite
- Docker / Docker Compose

## 機能
- ユーザー登録
- ログイン / ログアウト
- 投稿機能
- 投稿削除（自分の投稿のみ）
- 投稿にユーザー名表示

## 操作手順
#### EC2の環境に入っている前提
#### １.GithabからCloneをする
#### git clone https://github.com/nakamura-sekai/WebBoard.git
#### ２.コマンドでフォルダー移動（PowerShellを使用）
#### cd WebBord
#### ３.composeをする
#### sudo docker-compose up -d
#### ４.ブラウザで起動
#### EC2のパブリックIPをブラウザに貼り付ける

## 工夫した点
#### ・ログイン状態の管理
#### 　　セッションを使用してログイン状態を管理し、未ログインユーザーは自動的にログイン画面へリダイレクトするようにしました。
#### ・投稿の権限制御
#### 　　投稿にユーザーIDを紐づけ、自分の投稿のみ削除できるようにすることで簡易的なアクセス制御を実装しました。
#### ・UIの改善
#### 　　シンプルながらも見やすいデザインを意識し、ログイン・登録・掲示板の画面を統一したレイアウトにしました。
#### ・Dockerによる環境構築
#### 　　開発環境に依存せず、どの環境でも同じように動作するようDockerとDocker Composeを使用しました。

## 苦労した点
#### ・削除機能が動作しない問題
#### 　　GETとPOSTの違いや、フォームの書き方によって削除ができない問題が発生しました。
#### 　　最終的にPOSTメソッドで処理するように修正しました。
#### ・Docker環境構築
#### 　　DockerやComposeのバージョン違いによるコマンドエラーや、ポート設定（80番）の制限に苦労しました。
#### ・サーバー公開時の接続エラー
#### 　　EC2で「ページに到達できません」という問題が発生し、ポート設定やファイアウォールの確認を行って解決しました。
